The piano can be played via both clicking and the keyboard.

Keyboard row "qwertyuiop" corresponds to the white keys.

Keyboard row "asdfghjkl;'" corresponds to the black keys.

Keyboard row "zxcvbnm,./" corresponds to the white keys again.

To lower or raise octaves, use the ← and → arrow keys.
